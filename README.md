# Proyecto Acom. Diseño

Colección de implementacion de código enfocadas en patrones de diseño.  
Cada ejercicio presenta un problema real de código que puede ser resuelto aplicando un patrón de diseño específico.

## Objetivo

El objetivo de el proyecto es practicar la identificación y aplicación de patrones de diseño en situaciones concretas.

Cada kata incluye:

- **Contexto**: Un escenario real del mundo del desarrollo.

## Estructura

El código está organizado por carpetas, donde cada una contiene un ejercicio autocontenido con su propia documentación detallada.

## Patrones de Diseño Implementados

| Patrón | Tipo | Descripción del Ejercicio |
| :--- | :--- | :--- |
| **[Builder](./builder)** | Creacional | Sistema de construcción de hamburguesas complejas evitando constructores telescópicos. |
| **[Strategy](./strategy)** | Comportamiento | Carrito de compras con múltiples métodos de pago intercambiables (Tarjeta, PayPal). |
| **[Decorator](./decorator)** | Estructural | Sistema de cafetería para agregar ingredientes extra (leche, azúcar) dinámicamente. |
| **[Facade](./facade)** | Estructural | Un menú simplificado que coordina la creación de pedidos complejos para el cliente. |
| **[Template Method](./template_method)** | Comportamiento | Algoritmo estandarizado para preparar bebidas calientes (Café y Té) reutilizando pasos comunes. |
